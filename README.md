  https://stockfishchess.org/download/

Compiling instructions :macOSX 10.15:
```
    export SDKROOT=/Library/Developer/CommandLineTools/SDKs/MacOSX10.15.sdk
    uname -a
    make build ARCH=x86-64 COMP=clang
```
